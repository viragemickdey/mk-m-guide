# mk-m-guide

## 项目简介

mk-m-guide 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不绑定任何特定域名或网站，仅作为静态页面的合集存放与展示。

## 目录结构

```
mk-m-guide/
├── index.html          # 仓库主页（可选）
├── pages/              # 归档的 HTML 页面存放目录
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── assets/             # 公共资源（CSS、JS、图片等）
    ├── style.css
    └── ...
```

- **pages/**：存放所有独立的 HTML 页面，每个文件均可单独访问。
- **assets/**：存放页面所需的公共静态资源，便于统一管理。

## 页面归档说明

- 所有页面均为独立 HTML 文件，不依赖外部数据库或后端服务。
- 页面内容可能涉及学习笔记、工具页面、演示页面等，请根据具体文件判断用途。
- 归档页面不保证实时更新，部分页面可能为历史版本或实验性内容。
